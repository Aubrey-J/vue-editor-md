# 安装
npm i vue-editormd-wrap
# 使用
main.js
```JavaScript
import vueEditorMdWrap from 'vue-editormd-wrap'

Vue.use(vueEditorMdWrap)
```
vue page
```javascript
<editorMdWrap></editorMdWrap>
```
